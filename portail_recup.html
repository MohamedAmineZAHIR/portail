Description

CHRONOS est un portail de Production qui permet :
  • de produire des services internet de manière ordonnancée.
  • de réaliser la configuration des services sur les plateformes de services
  • de tenir à jour un référentiel de données techniques
  • d'être interfacé avec d’autres SI tels que OPERA, OPUS
  • d'effectuer des opérations de maintenance sur les produits gérés et installés(SAV)
